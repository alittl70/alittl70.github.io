Safari Destination Vote

The pride is planning its next great migration — but where should we go? The Safari Destination Vote is a polling tool that lets adventurers cast their vote for the next expedition destination.

Each traveler gets one vote per destination, and the system keeps things fair using JavaScript's Map and Set data structures. No double voting, no duplicate entries — just clean, honest democracy on the savanna.

Current destinations up for vote include the sweeping plains of the Serengeti, the misty mountains of Morocco, and the vibrant streets of Istanbul. Cast your vote and let the journey begin.

Built with JavaScript Map and Set as part of the FCC JavaScript Certification.
